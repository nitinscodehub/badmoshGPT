# 🔥 BadmoshGPT CLI

**BadmoshGPT CLI** ek powerful terminal-based AI chatbot hai jo OpenRouter API ke through LLMs se baat karta hai. Isme file upload, shell command execution, browser control, session save/load, aur bahut kuch hai.

## 🚀 Features

| Feature | Description |
|---|---|
| 💬 **Chat with AI** | Multiple models support (DeepSeek, MiniMax, Qwen, etc.) |
| 🖥️ **Shell Execution** | `!ls`, `!nmap`, `!sqlmap` — run any terminal command |
| 🌐 **Browser Control** | `%open https://...` or `%search query` |
| 📁 **File Upload** | Upload images, PDFs, text files and ask AI about them |
| 📝 **File Write/Read** | `@write path` + content + `EOF` to create files |
| 🎯 **Payload Templates** | 10 pre-built hacking/tool prompts |
| 🔄 **Multi-Model Attack** | Same query to 3 models simultaneously |
| 🌍 **Web Search** | Toggle web search for real-time info |
| 💾 **Session Save/Load** | Save chat history and resume later |
| 🔧 **Customizable** | Change model, API key, system prompt anytime |
| 🚀 **System-wide Command** | `badmoshgpt` from any terminal |
| 🎨 **Hindi+English Gaali** | Desi style mein baat karta hai 😈 |

## 📦 Installation

### 1. Clone the repo
```bash
git clone https://github.com/nitinscodehub/BadmoshGPT
cd BadmoshGPT
```

### 2. Setup venv (optional but recommended)
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install requests pyfiglet langdetect
```

### 4. Get an API key
- Go to [OpenRouter.ai](https://openrouter.ai/keys)
- Sign up and create a free API key

### 5. Set your API key
```bash
python3 ai.py
```
Menu se option **3** select karke API key daalo, ya direct `wormgpt_config.json` edit karo:
```json
{
  "api_key": "sk-or-v1-your-key-here",
  "base_url": "https://openrouter.ai/api/v1",
  "model": "deepseek/deepseek-chat",
  "language": "English"
}
```

## 🚀 Usage

### Run from anywhere (recommended)
```bash
badmoshgpt
```

### Or run directly
```bash
python3 ai.py
```

### System-wide install
```bash
mkdir -p ~/.local/bin
cat > ~/.local/bin/badmoshgpt << 'EOF'
#!/bin/bash
exec python3 /full/path/to/BadmoshGPT/ai.py "$@"
EOF
chmod +x ~/.local/bin/badmoshgpt
```
Ab bas `badmoshgpt` type karo kisi bhi terminal mein!

## 🎯 Commands (Chat ke andar)

| Command | Example | Kaam |
|---|---|---|
| `!<shell>` | `!ls -la` | Shell command execute kare |
| `%open <url>` | `%open https://google.com` | Browser mein URL khole |
| `%search <q>` | `%search kali linux tools` | Google search kare |
| `@write <path>` | `@write hack.py` | File create kare (content daalo, phir `EOF`) |
| `@read <path>` | `@read /etc/passwd` | File content dekhe |
| `upload <path>` | `upload image.jpg` | File attach kare (image/PDF/text) |
| `payloads` | `payloads` | Pre-built templates show kare |
| `multi <query>` | `multi wifi hack` | 3 models ek saath query kare |
| `web` | `web` | Web search toggle (ON/OFF) |
| `save` | `save` | Chat session save kare |
| `load` | `load` | Saved session load kare |
| `new` | `new` | Chat reset kare |
| `menu` | `menu` | Main menu mein jaaye |
| `help` | `help` | Commands list dekhe |
| `exit` | `exit` | Band kare |

## 🧠 Available Models

Menu option **2** se change kare:

- `deepseek/deepseek-chat` — Best for uncensored (recommended)
- `minimax/minimax-m3` — Multimodal (image+text)
- `qwen/qwen3.5-plus` — Strong reasoning
- Custom model ID bhi daal sakte hain
